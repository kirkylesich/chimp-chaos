Chimp Chaos Operator
====================

Локальная сборка и тесты без внешних сервисов. Адаптеры по умолчанию безопасно отключены.

Требования
----------
- Rust stable (см. rust-toolchain.toml)
- make

Команды
-------
```
make fmt
make lint
make test
make build
make run
```

Структура Workspace
-------------------
См. `Cargo.toml` workspace members.
